=====================================
Complete Installation Guide
=====================================

This is a detailed guide for the installation of PySPH. The guide
covers installation from scratch without the assumption of
administrative rights on your system.

PySPH is intended to be cross platform and is actively built and
tested on Windows and Linux. Look at the appropriate guide for your
system.
